Capstone - Module 1 Collecting Data Using APIs

## Project Overview
This project demonstrates how to collect job data using APIs, process JSON responses, analyze job counts by city and technology, and export results to Excel.

## Skills Demonstrated
- API requests using Python (requests)
- JSON parsing
- Data filtering and counting
- Pandas DataFrame creation
- Writing results to Excel
- Basic data analysis workflow

## Technologies Used
- Python
- Pandas
- Requests
- Openpyxl

## Key Analysis
- Counted job postings by city (e.g., Atlanta, Los Angeles)
- Counted job postings by technology (Python, SQL, etc.)
- Exported structured results to Excel

## Output
Generated Excel file containing technology job counts.
